1. PENGENALAN APLIKASI
================================================================================

RamStore adalah aplikasi marketplace penjualan online yang memungkinkan:
- Customer untuk melihat dan membeli produk
- Admin untuk mengelola produk (tambah, edit, hapus)
- Integrasi dengan WhatsApp untuk pesanan

3. PANDUAN PENGGUNA (CUSTOMER)
================================================================================

A. MELIHAT PRODUK
   1. Buka halaman utama (index.html)
   2. Scroll ke bagian "Produk"
   3. Produk akan ditampilkan dalam bentuk kartu (card)
   4. Setiap kartu menampilkan:
      - Gambar produk
      - Nama produk
      - Harga
      - Deskripsi singkat
      - Tombol "Tambah"

B. FILTER PRODUK BERDASARKAN KATEGORI
   1. Di halaman utama, lihat menu kategori di atas daftar produk
   2. Pilih kategori yang diinginkan:
      - "Semua" - menampilkan semua produk
      - "Electronics" - produk elektronik
      - "Accessories" - produk aksesori
      - (kategori lainnya sesuai data)
   3. Produk akan difilter otomatis sesuai kategori

C. MELIHAT DETAIL PRODUK
   1. Klik pada kartu produk atau nama produk
   2. Anda akan membuka halaman detail produk
   3. Halaman detail menampilkan:
      - Gambar produk (besar)
      - Nama produk
      - Harga lengkap
      - Rating bintang
      - Kategori
      - Deskripsi lengkap
      - Informasi pengiriman
      - Produk terkait (rekomendasi)

D. MENAMBAH PRODUK KE KERANJANG
   Cara 1: Dari halaman utama
   - Klik tombol "Tambah" pada kartu produk
   - Produk akan ditambahkan ke keranjang
   - Tombol akan berubah menjadi "✓ Ditambahkan" (2 detik)
   
   Cara 2: Dari halaman detail produk
   - Tentukan jumlah produk (default: 1)
   - Klik tombol "Tambah ke Keranjang"
   - Pesanan akan ditambahkan ke keranjang

E. MELIHAT KERANJANG BELANJA
   1. Klik ikon keranjang di navbar (atas kanan)
   2. Badge merah menunjukkan jumlah item di keranjang
   3. Anda akan diarahkan ke halaman checkout

F. MENGHAPUS PRODUK DARI KERANJANG
   1. Di halaman checkout, lihat daftar produk
   2. Klik "Hapus" pada produk yang ingin dihapus
   3. Produk akan langsung terhapus dari keranjang

4. PANDUAN CHECKOUT
================================================================================

A. MASUK KE HALAMAN CHECKOUT
   1. Klik ikon keranjang di navbar
   2. Atau buka URL: http://localhost/penjualan-app/checkout.html

B. DAFTAR PRODUK
   Halaman checkout menampilkan:
   - Gambar produk thumbnail
   - Nama produk
   - Jumlah x harga per item = subtotal
   - Tombol "Hapus" untuk setiap item

C. MEMILIH METODE PENGIRIMAN
   Pilih salah satu dari 3 opsi:
   1. Ambil di Toko (Gratis)
      - Tidak ada biaya pengiriman
      - Waktu: Langsung setelah pembayaran
   
   2. Pengiriman Standar (Rp 10.000)
      - Waktu: 1-3 hari kerja
      - Cocok untuk pengiriman jarak jauh
   
   3. Pengiriman Express (Rp 25.000)
      - Waktu: Same-day (hari yang sama)
      - Untuk pesanan yang mendesak

D. MENGISI DATA PENGIRIMAN
   Lengkapi data berikut (WAJIB):
   1. Nama Lengkap
      - Contoh: Rama Raditya
   
   2. Nomor Telepon
      - Contoh: 0812345678
      - Gunakan nomor yang aktif
   
   3. Alamat Lengkap (WAJIB)
      - Contoh: Jl. Ahmad Yani No. 123, Kelurahan Matahari, Kecamatan Jaya, Kota Surabaya, Jawa Timur 60111
      - Semakin detail semakin baik
   
   4. Catatan Tambahan (OPSIONAL)
      - Untuk instruksi khusus ke kurir/penjual
      - Contoh: "Tolong bunyikan bel pintu depan"

E. RINGKASAN PEMBAYARAN
   Halaman menampilkan:
   - Subtotal: Total harga produk
   - Pengiriman: Biaya berdasarkan metode pilihan
   - PPN (10%): Pajak pertambahan nilai
   - Total Pembayaran: Jumlah akhir yang harus dibayar

F. MENYELESAIKAN PESANAN
   1. Pastikan semua data sudah lengkap dan benar
   2. Klik tombol "Lanjutkan Pembayaran"
   3. WhatsApp akan terbuka otomatis dengan:
      - Data lengkap pesanan Anda
      - Detail produk
      - Total pembayaran
   4. Kirim pesan ke penjual
   5. Tunggu konfirmasi dari penjual
   6. Halaman akan kembali ke beranda

6. PANDUAN ADMIN
================================================================================

A. LOGIN KE HALAMAN ADMIN
   URL: http://localhost/penjualan-app/login.html
   
   Kredensial Demo:
   - Username: admin
   - Password: admin123
   
   Instruksi:
   1. Buka halaman login
   2. Masukkan username dan password
   3. Klik tombol login
   4. Jika berhasil, Anda akan diarahkan ke halaman kelola produk

B. HALAMAN KELOLA PRODUK
   URL: http://localhost/penjualan-app/products.html (setelah login)
   
   Menampilkan:
   - Tabel daftar semua produk
   - Kolom: ID, Gambar, Nama, Kategori, Harga, Aksi

C. MENAMBAH PRODUK BARU
   1. Klik tombol "Tambah Produk" (atas kanan)
   2. Modal form akan terbuka
   3. Isi data produk:
      
      a. Nama Produk
         - Nama unik produk
         - Contoh: "Smartphone Samsung Galaxy A12"
      
      b. Harga
         - Harga dalam rupiah (angka)
         - Contoh: 2500000
      
      c. Kategori
         - Pilih dari dropdown atau ketik kategori baru
         - Contoh: "Electronics", "Accessories"
         - Kategori akan dinormalisasi (jewelry → accessories)
      
      d. URL Gambar
         - Link gambar produk (format URL)
         - Contoh: https://example.com/gambar-produk.jpg
      
      e. Deskripsi
         - Penjelasan lengkap produk
         - Contoh: "Smartphone terbaru dengan prosesor Helio G25..."
   
   4. Klik tombol "Simpan"
   5. Produk akan ditambahkan ke database
   6. Notifikasi sukses akan muncul

D. MENGEDIT PRODUK EXISTING
   1. Klik tombol "Edit" (icon pensil) pada baris produk
   2. Modal form akan terbuka dengan data produk terpilih
   3. Ubah data yang diinginkan
   4. Klik tombol "Update"
   5. Produk akan diperbarui
   6. Notifikasi sukses akan muncul

E. MENGHAPUS PRODUK
   1. Klik tombol "Hapus" (icon trash) pada baris produk
   2. Konfirmasi dialog akan muncul: "Apakah Anda yakin ingin menghapus produk ini?"
   3. Klik "OK" untuk confirm, atau "Cancel" untuk batal
   4. Produk akan dihapus dari database
   5. Notifikasi sukses akan muncul

F. LOGOUT
   1. Klik link "Logout" di navbar (atas kanan)
   2. Anda akan diarahkan kembali ke halaman login
   3. Semua sesi admin akan dihapus

G. NOTIFIKASI
   - Sukses (Hijau): Operasi berhasil dilakukan
   - Error (Merah): Ada kesalahan saat operasi
   - Notifikasi akan hilang otomatis setelah 5 detik

8. TROUBLESHOOTING
================================================================================

MASALAH: "Halaman tidak memuat"
SOLUSI:
- Pastikan URL benar: http://localhost/penjualan-app/
- Cek koneksi internet
- Refresh halaman (Ctrl+F5)
- Bersihkan cache browser

MASALAH: "Login admin gagal"
SOLUSI:
- Pastikan username dan password benar (admin/admin123)
- Bersihkan cache browser
- Coba mode private/incognito
- Refresh halaman

MASALAH: "Produk tidak muncul di keranjang"
SOLUSI:
- Refresh halaman
- Bersihkan cache browser dan localStorage
- Coba browser lain

MASALAH: "WhatsApp tidak terbuka saat checkout"
SOLUSI:
- Pastikan WhatsApp sudah install
- Periksa pop-up blocker browser
- Izinkan akses pop-up untuk website ini
- Coba manual: salin pesan, buka WhatsApp, paste pesan

MASALAH: "Gambar produk tidak muncul"
SOLUSI:
- Cek URL gambar (harus valid dan accessible)
- Gunakan format image: jpg, png, gif, webp
- Pastikan link tidak expired
- Coba upload gambar ke server sendiri

MASALAH: "Kategori tidak muncul di filter"
SOLUSI:
- Pastikan sudah ada produk dengan kategori tersebut
- Refresh halaman
- Tambah produk baru dengan kategori yang ingin ditampilkan

B. STRUKTUR FOLDER

   ├── login.html              (Halaman login admin)
   ├── products.html           (Halaman kelola produk)
   ├── product-detail.html     (Halaman detail produk)
   ├── checkout.html           (Halaman checkout)
   ├── alamat.html             (Halaman alamat)
   ├── kontak.html             (Halaman kontak)
   ├── tentang.html            (Halaman tentang)
   ├── admin.html              (Redirect ke products.html)
   ├── script.js               (JavaScript utama)
   ├── login.js                (Logic login)
   ├── products.js             (Logic CRUD produk)
   ├── product-detail.js       (Logic detail produk)
   ├── checkout.js             (Logic checkout)
   ├── page-loader.js          (Loading indicator)
   ├── style.css               (Custom CSS)
   ├── footer.html             (Template footer)
   ├── api/
   │   ├── api.php             (API backend)
   │   └── db.php              (Koneksi database)
   └── img/                    (Folder gambar)

C. DATABASE
   Nama: penjualan_db
   User: root (no password)
   Host: localhost
   
   Tabel: products
   Kolom:
   - id (INT, Primary Key, Auto Increment)
   - title (VARCHAR, Nama produk)
   - price (DECIMAL, Harga)
   - description (TEXT, Deskripsi)
   - category (VARCHAR, Kategori)
   - image (VARCHAR, URL gambar)

D. API ENDPOINTS
   GET /api/api.php?action=get_products
   - Mengambil semua produk
   
   POST /api/api.php?action=add_product
   - Menambah produk baru
   - Body: {title, price, category, image, description}
   
   POST /api/api.php?action=update_product
   - Mengupdate produk
   - Body: {id, title, price, category, image, description}
   
   POST /api/api.php?action=delete_product
   - Menghapus produk
   - Body: {id}

E. LOCAL STORAGE KEYS
   - cart: Menyimpan keranjang belanja
   - orders: Menyimpan riwayat pesanan
   - adminToken: Token login admin
   - adminUsername: Username admin yang login

F. KONFIGURASI PENTING
   - Nomor WhatsApp: Di file checkout.js (const WHATSAPP_NUMBER)
   - Demo Akun Admin: Di file login.js (const DEMO_CREDENTIALS)
   - Bootstrap CDN: Di file HTML (jika perlu offline, download dan letakkan lokal)

10. FAQ (FREQUENTLY ASKED QUESTIONS)
================================================================================

Q: Bagaimana cara mengubah nomor WhatsApp penjual?
A: Buka file checkout.js, cari "const WHATSAPP_NUMBER", ubah nomor ke nomor Anda
   Format: '62' + nomor tanpa '0' (contoh: '6281234567890')

Q: Bagaimana cara mengubah password admin?
A: Buka file login.js, cari "const DEMO_CREDENTIALS", ubah password di field password

Q: Bagaimana cara backup database?
A: Gunakan PHPMyAdmin atau export database MySQL secara manual

Q: Apakah bisa menambah lebih dari 1 akun admin?
A: Saat ini hanya support demo credentials. Untuk multi-user, perlu modifikasi backend

Q: Bagaimana cara membayar pesanan?
A: Semua pembayaran diatur langsung via WhatsApp antara customer dan penjual

Q: Apakah ada sistem tracking pesanan?
A: Tidak ada sistem tracking terintegrasi. Tracking bisa diatur via WhatsApp

Q: Bagaimana cara reset keranjang customer?
A: Bersihkan localStorage di browser (F12 > Application > Local Storage)

Q: Bagaimana cara export data pesanan?
A: Data pesanan disimpan di localStorage customer. Untuk centralized, perlu database backend
